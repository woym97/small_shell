[DESCRIPTION]
This program can be run via the command line and will process the movies in the file via the command line. An example file is provided in this directory to provide an example of the required formatting. 

[TO RUN]
To run this program, extract the files from the zip drive and place the CSV file in the same directory as the files themselves. Once in the program directory via command line run the following commands to run the program:

	gcc --std=gnu99 -o movies main.c
	./movies movies_sample_1.csv

Replace "movies_sample_1.csv" with the desired file input
